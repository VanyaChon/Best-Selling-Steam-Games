Steam Games Analyzer

Эта модель машинного обучения предсказывает, станет ли игра в Steam бестселлером на основе ключевых характеристик игры. 
Модель использует алгоритм классификации Random Forest для анализа характеристик игр и предоставляет прогнозы с точностью приблизительно 77-80%.
В модель встроена балансировка классов для обработки естественного дисбаланса между бестселлерами и обычными играми. 
Модель использует четыре основных признака для прогнозирования: цена игры, год выпуска, основной жанр и информация о разработчике.

Метрики качества
Accuracy: 77-80%
Precision: ~0.78
Recall: ~0.76
F1-Score: ~0.77

Что умеет модель
 Анализирует игры по 4 параметрам: цена, год выпуска, жанр и разработчик
 Находит похожие игры в базе данных
 Работает через удобный веб-интерфейс

! Как это работает !
Ввод данных → Вы указываете параметры игры
Анализ → Модель обрабатывает информацию
Результат → Получаете прогноз и рекомендации
